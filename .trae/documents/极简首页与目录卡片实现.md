## 目标与范围
- 在项目根新增 `index.html` 作为首页，展示一组卡片目录，跳转至各详情页（当前包含 `investment/voo_vs_spy/voo_vs_spy/voo_vs_spy.html`）。
- 保持“单文件、零构建”的极简实现，交互简洁优雅、移动端友好。
- 技术栈与视觉风格对齐 `investment` 目录页面。

## 技术栈对齐
- 使用 `Tailwind CSS` CDN：`https://cdn.tailwindcss.com`。
- 使用 `Font Awesome` CDN（与详情页一致的 6.x 链接）。
- 原生 JS（不引入框架、不依赖构建）。
- Tailwind `tailwind.config` 内联复用 `colors.primary/secondary` 等扩展，统一配色与字体。

## 信息架构与 UI
- 顶部导航：站点名与简要说明；可选按钮返回首页或跳转到分类锚点。
- 内容区：响应式网格卡片（标题、简介、标签、更新时间），卡片 hover 动效与轻量阴影，点击进入详情页。
- 搜索与过滤：页内输入框实现前端关键词过滤（标题/描述/标签），不依赖后端。
- 无 JS 回退：同时渲染一个简洁的链接列表，保障禁用 JS 时仍可跳转。
- 页脚：版权与声明，风格与详情页保持一致。

## 数据来源与扩展方式
- 首页内置 `cards` 数组（硬编码在页面中，保证单文件可用）：`{ title, desc, href, category, tags, updatedAt }`。
- 现有条目：
  - `title`: "VOO 与 SPY 深度对比"
  - `desc`: "长期买入持有策略的最佳选择"
  - `href`: `investment/voo_vs_spy/voo_vs_spy.html`
  - `category`: "Investment"
  - `tags`: ["ETF", "S&P500", "VOO", "SPY"]
  - `updatedAt`: "2025-11"
- 后续新增详情页时，仅需在首页数组中追加一项即可。

## 交互设计
- 关键词搜索：输入后即时过滤卡片，支持拼音/英文关键词的简单匹配（标题、描述、标签）。
- 分类过滤（可选）：标签按钮切换视图。
- 键盘导航：卡片可聚焦，方向键移动、`Enter` 打开链接。
- 轻量动画：卡片进入时渐显与位移，尊重 `prefers-reduced-motion`。

## 无障碍与性能
- 语义化结构（`header/main/footer`、`nav`、`section`）。
- 可达性：`tabindex`、可见焦点、`aria-label` 与按钮描述。
- SEO：`<meta>`、合理的标题层级、可抓取的链接列表。
- 性能：尽量少用阴影与复杂滤镜，避免不必要脚本；图片图标统一用 `Font Awesome`。

## 路径与结构
- 新增文件：`/index.html`（项目根）。
- 卡片链接：`investment/voo_vs_spy/voo_vs_spy.html` 等现有/新增详情页面。

## 验证与测试
- 本地直接打开 `index.html`，确认卡片样式、搜索过滤、键盘导航均正常。
- 点击卡片跳转到现有详情页可用（含移动端与桌面响应式检查）。
- 禁用 JS 时，链接列表仍可跳转。

## 交付物
- 单文件首页：`index.html`，使用 Tailwind CDN + Font Awesome，内联 JS/数据。
